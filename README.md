# Password Strength Checker 🔐

A beginner cybersecurity project built with Python that analyzes how strong or weak a password is.

## What it checks
- Minimum length of 8 characters
- Uppercase and lowercase letters
- Numbers and special characters
- Sequential patterns like 123 or abc

## How to run
1. Make sure Python is installed
2. Download password_checker.py
3. Open terminal and run:
python password_checker.py
4. Enter any password and get instant feedback

## Skills used
- Python fundamentals
- ASCII logic with ord()
- String slicing and loops
- Security thinking
